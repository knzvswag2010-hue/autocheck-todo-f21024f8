## Инструкция
1. Склонируйте репозиторий
2. Создайте файл с решением (`solution.py` или соответствующий вашему языку)
3. Запустите проверку через `git push`
4. Результаты проверки появятся во вкладке Actions

## Проверка
После каждого push автоматически запускаются тесты. Результаты можно увидеть в GitHub Actions.
